## 产品概述

为 Edge 浏览器新标签页的「设置」侧边栏进行视觉与交互优化，使其更现代、易用、层次清晰。

## 核心功能

- 优化抽屉内所有按钮的视觉表现（关闭按钮、折叠 ▶、主操作按钮、危险按钮）
- 增强悬停（hover）、按下（active）、聚焦（focus）三种状态的视觉反馈
- 调整 section / row 间距、对齐与层级关系，提升呼吸感
- 强化单选源（导入 / 内置）选中态辨识度
- 优化滑块、颜色选择器、预览网格的交互细节
- 添加入场过渡与按钮微动效

## 视觉边界

- 抽屉宽度 360px，遵循现有 8–10px 圆角体系
- 保持蓝色主色 #4285f4、红色危险色 #d93025、灰阶文字与半透明白底
- 折叠区域展开 / 收起动画继续使用现有 0.3s 过渡节奏

## 技术栈

- 项目类型：Edge 浏览器新标签页扩展（纯 HTML / CSS / 原生 JS）
- 修改范围：仅 `d:\MrKLLM\MyEdgeNewTab\styles.css`
- 不改动 `index.html` 结构、不改动 `script.js` 逻辑
- 复用现有 CSS 变量与 `backdrop-filter` 毛玻璃质感

## 实施策略

- 在 `:root` 中补充交互系统变量（按钮悬停 / 按下偏移、过渡曲线、聚焦环、section 间距）
- 按「按钮 → 单选源 → 排版层级 → 控件反馈 → 入场动画」由上到下覆盖，避免冲突
- 沿用现有 0.2s ease 过渡节奏统一所有 hover / active 反馈
- 新增抽屉 slide-in 入场动画（仅 transform / opacity，不触发重排）
- 主按钮增加 `translateY(-1px)` 浮起 + active 复位 1px 反馈
- 危险按钮在 hover 时提升红色饱和度 + 轻微外阴影警示
- 单选源使用 4px 左侧色条 + 浅蓝底色双指示，替代纯文字加粗
- 折叠 ▶ 按钮在 expanded 时旋转 90°，hover 显示圆形微高亮

## 实施要点

- 性能：仅在 transform / opacity / box-shadow / background 上做过渡，不触发布局重排
- 兼容性：沿用浏览器扩展上下文（Chromium 内核），无需添加前缀
- 影响范围控制：仅改 `.settings-*` 选择器，不污染背景轮播与搜索框
- 渐变阴影：使用 `box-shadow: 0 1px 2px rgba(...), 0 4px 12px rgba(...)` 提升层次但不浮夸

## 设计思路

基于「轻量毛玻璃 + 现代极简」风格，在现有 360px 抽屉框架内做精细化打磨。整体目标是：层级更清晰、操作更有反馈、阅读更舒适。

## 页面规划

仅 1 个核心面板：设置抽屉（drawer），包含 5 个 section

- 背景图片（默认展开）
- 搜索框外观（折叠）
- 搜索按钮外观（折叠）
- 轮播（折叠）
- 其他操作（默认展开）

## 单页区块设计（自顶向下）

1. **抽屉 Header**（高 52px）：左侧 16px / 600 标题「设置」，右侧 32×32 圆形热区关闭按钮，hover 浅灰圆底
2. **Section 容器**（上下 padding 16px）：标题 13px / 600 灰色，标题与内容间距 12px；section 之间通过 1px 浅色分割线分隔
3. **背景图片 Section**：单选源为卡片式（4px 蓝色左条 + 浅蓝底 + 加粗文本），已导入行采用「标签 — 操作」两列对齐，「+ 选择本地图片」为主操作蓝按钮
4. **可折叠 Section Header**：整行可点击（不仅是 ▶ 按钮），hover 显示浅灰底，▶ 旋转 90° 表达展开
5. **设置项 Row**（三列 grid 1fr 140px 64px）：左 label 12px / 500 灰黑，中间控件铺满，右值文本等宽数字右对齐，row 间距 14px
6. **其他操作 Section**：「恢复默认配置」全宽主蓝按钮，与「+ 选择本地图片」视觉一致

## 交互反馈

- 主蓝按钮：默认蓝底浅蓝边 → hover 蓝底加深 + 上浮 1px + 阴影增强 → active 下沉 1px + 阴影收敛
- 危险按钮：默认浅红 → hover 红底加深 + 红色外阴影 → active 下沉 1px
- 关闭按钮：hover 圆形浅灰底 + 旋转 90° ✕，active 略深
- 单选源：hover 浅蓝底，选中 4px 蓝左条 + 浅蓝底 + 加粗
- 滑块 / 颜色：focus 显示 2px 蓝色聚焦环

## 入场动画

抽屉整体从右侧 `translateX(20px)` 滑入 + 透明度 0→1，250ms cubic-bezier(0.2, 0.8, 0.2, 1)；遮罩淡入 200ms
